# Documentación del Proyecto Mokepon

Esta carpeta reúne la documentación técnica correspondiente al desarrollo del proyecto **Mokepon**.

Con el objetivo de mejorar la organización del repositorio, la documentación fue dividida en diferentes etapas del desarrollo. Cada documento agrupa los conceptos estudiados durante una fase específica del proyecto, facilitando tanto la lectura como el mantenimiento de la información.

---

# Tabla de Contenido

## 1. Maquetación de Páginas Web para Videojuegos en HTML y JavaScript

Archivo:

```
01-maquetacion-html-js.md
```

Documenta la construcción inicial del videojuego utilizando HTML y JavaScript.

En esta etapa se desarrolla la estructura de la interfaz, la selección de mascotas, la manipulación del DOM, el manejo de eventos, la lógica del combate y los fundamentos necesarios para construir una aplicación web interactiva.

Temas principales:

- HTML
- JavaScript
- DOM
- Eventos
- Lógica del juego
- Manipulación dinámica del documento

---

## 2. Estilos con CSS

Archivo:

```
02-estilos-css.md
```

Reúne toda la documentación relacionada con la construcción visual del proyecto.

Durante esta etapa se implementan estilos utilizando CSS, Flexbox, Grid, Media Queries y pseudoclases para construir una interfaz moderna, organizada y adaptable a diferentes dispositivos.

Temas principales:

- CSS
- Selectores
- Flexbox
- Grid
- Responsive Design
- Media Queries
- Modelo de Caja
- Botones
- Layout

---

## 3. Optimización del Código

Archivo:

```
03-optimizacion-codigo.md
```

Describe el proceso de refactorización y mejora de la arquitectura del proyecto.

En esta fase se reorganiza la lógica del videojuego mediante clases, objetos, arreglos y funciones reutilizables, buscando un código más limpio, escalable y fácil de mantener.

Temas principales:

- Clases
- Objetos
- Arrays
- Funciones
- Refactorización
- Programación Orientada a Objetos
- Renderizado dinámico
- Optimización del código
