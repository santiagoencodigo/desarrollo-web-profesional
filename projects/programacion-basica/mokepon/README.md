# Mokepon

Esta carpeta reúne la documentación técnica y los apuntes relacionados con el desarrollo del proyecto **Mokepon**, un videojuego desarrollado como parte del proceso de aprendizaje de Programación Básica.

A lo largo del proyecto se aplican progresivamente conceptos fundamentales de HTML, CSS y JavaScript, permitiendo comprender el desarrollo completo de una aplicación web interactiva. Durante su construcción se trabajan temas como la manipulación del DOM, eventos, programación orientada a objetos, renderizado dinámico, estilos con CSS, optimización del código y organización de proyectos.

Además de servir como evidencia del proceso de aprendizaje, esta documentación funciona como material de consulta para comprender cómo evolucionó el proyecto desde una estructura básica hasta una aplicación web más organizada y mantenible.
