# Documentación — Desarrollo Web Profesional

Bienvenido a la sección de documentación de mi viaje de aprendizaje en desarrollo web. Aquí encontrarás todos los apuntes, resúmenes y reflexiones que he creado mientras construyo los proyectos de este portafolio.

> **Propósito:** Esta carpeta contiene los fundamentos teóricos que sustentan mi formación en desarrollo web. Cada documento aborda un concepto esencial — desde la historia de Internet hasta el control de versiones — y está diseñado para servir como material de consulta y referencia en mi proceso de aprendizaje.
